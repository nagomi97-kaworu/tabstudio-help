# TAB Studio Help (English)

User help and support hub for TAB Studio.

TAB Studio is a desktop app for viewing, editing, and playing back TAB notation
(guitar, bass, etc.). Automatic AI transcription from video/YouTube/audio is a
**planned feature** — see [Getting Started](docs/en/getting-started.md) for what's
available today.

> This repository is for user support only. The application's source code is not
> included here.

## Documentation

- [Getting Started](docs/en/getting-started.md)
- [FAQ](docs/en/faq.md)
- [Troubleshooting](docs/en/troubleshooting.md)

## Reporting bugs / requesting features

In the app menu, go to **Help → Report a Bug / Send Feedback**. A ready-made Issue
template (including your environment info) is generated and copied to your clipboard.

Paste it into this repository's [Issues](../../issues).

To create an Issue directly, use one of these templates:

- [Bug report](../../issues/new?template=bug_report.md)
- [Feature request](../../issues/new?template=feature_request.md)

## Support policy

- This repository accepts **usage questions** and **bug reports / feature requests**
  for TAB Studio
- We may not be able to answer questions about internal implementation details

# TAB Studio ヘルプ(日本語)

TAB Studio(タブスタジオ)のユーザー向けヘルプ・サポート窓口リポジトリです。

TAB Studio は、ギター/ベース等のTAB譜を表示・編集・再生できるデスクトップアプリです。
動画・YouTube・音声からのAI自動採譜は**将来機能(開発中)**です。現在使える機能については
[使い方ガイド](docs/ja/getting-started.md)をご覧ください。

## 不具合の報告・要望の送信

アプリ内メニューの **ヘルプ → 不具合を報告 / 要望を送る** から、環境情報を含む
Issue用のテンプレートが自動生成されます(クリップボードにコピーされます)。

そのままこのリポジトリの [Issues](../../issues) にペーストして投稿してください。

直接 Issue を作成する場合は、以下のテンプレートを使ってください:

- [不具合報告](../../issues/new?template=bug_report.md)
- [要望](../../issues/new?template=feature_request.md)
