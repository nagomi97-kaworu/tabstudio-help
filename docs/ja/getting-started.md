# 使い方ガイド

TAB Studio の基本的な操作の流れを説明します。

> **動画・YouTube・音声からのAI自動採譜機能は現在開発中の将来機能です。**
> 以下は既存のTABファイルを読み込んで使う、現時点で利用できる操作の流れです。

## 1. 曲を読み込む

メニュー「ファイル」から操作します。

- **TABファイルを開く**: 既存のGuitar Pro形式(.gp/.gp5等)のファイルを開く
- **プロジェクトを開く**: 以前保存したプロジェクト(.tabsp)を開く

## 2. TAB譜を確認・再生する

画面中央にTAB譜(五線譜/タブ譜)が表示されます。

- 再生ボタン(またはスペースキー)で再生/停止
- 再生速度スライダーで再生速度を変更可能
- 譜面をドラッグすると選択範囲がループ再生対象になります

## 3. トラックミキサーで音量・音源を調整する

画面右側の「トラックミキサー」タブで、トラックごとに以下を操作できます:

- M(ミュート)/ S(ソロ)
- 音量(Vol)/ パン(Pan)
- 音源(VSTi)の割り当て
- エフェクト(FX)の追加(最大3つ)
- 五線譜/TAB譜/符幹の表示切り替え

「全体ミキサー」タブでは、曲全体・動画音声・マスター音量をまとめて調整できます。

## 4. プロジェクトを保存する

メニュー「ファイル → プロジェクトを保存」で、TAB譜・ミキサー設定・音源の割り当て
などをまとめて保存できます。次回「プロジェクトを開く」で同じ状態から再開できます。

## 今後の予定(開発中の機能)

- 動画ファイル/YouTube URLからの音声取り込み
- AIによる自動採譜(最小音符の長さ・クオンタイズ・カポ位置・弦割り当ての調整を含む)
- 動画とTAB譜の同期再生

これらは順次公開予定です。進捗はこのリポジトリの更新でお知らせします。
