# 使い方ガイド

TAB Studio の基本的な操作の流れを説明します。

## 1. 曲を読み込む

以下のいずれかの方法で曲を読み込みます(メニュー「ファイル」から操作します)。

- **TABファイルを開く**: 既存のGuitar Pro形式(.gp/.gp5等)のファイルを開く
- **動画ファイルを開く**: 手元の動画ファイルから音声を取り込む
- **YouTube URLを開く**: YouTubeのURLを指定して音声を取り込む
- **プロジェクトを開く**: 以前保存したプロジェクト(.tabsp)を開く

## 2. AIで採譜する

動画/YouTube/音声を読み込んだ場合、画面右側の解析タブから「解析開始」を押すと、
AIによる自動採譜が実行されます。処理には曲の長さに応じて時間がかかります。

採譜結果は以下の項目で調整できます:

- 最小音符の長さ / クオンタイズ(細かい音の丸め方)
- カポの位置
- 弦割り当ての再調整

採譜が完了すると、自動的にTAB譜として画面に表示されます。

## 3. TAB譜を確認・再生する

画面中央にTAB譜(五線譜/タブ譜)が表示されます。

- 再生ボタン(またはスペースキー)で再生/停止
- 再生速度スライダーで再生速度を変更可能
- 譜面をドラッグすると選択範囲がループ再生対象になります

## 4. トラックミキサーで音量・音源を調整する

画面右側の「トラックミキサー」タブで、トラックごとに以下を操作できます:

- M(ミュート)/ S(ソロ)
- 音量(Vol)/ パン(Pan)
- 音源(VSTi)の割り当て
- エフェクト(FX)の追加(最大3つ)
- 五線譜/TAB譜/符幹の表示切り替え

「全体ミキサー」タブでは、曲全体・動画音声・マスター音量をまとめて調整できます。

## 5. プロジェクトを保存する

メニュー「ファイル → プロジェクトを保存」で、TAB譜・ミキサー設定・音源の割り当て
などをまとめて保存できます。次回「プロジェクトを開く」で同じ状態から再開できます。
