# トラブルシューティング

## 起動しない / すぐに終了する

1. お使いのPCがWindows 10/11(64bit)であることを確認してください
2. 一度PCを再起動してから再度起動してみてください
3. 改善しない場合は「不具合を報告」から、発生状況(起動直後か、何か操作した後か等)を
   添えてご連絡ください。ログフォルダの内容を添付いただけるとより早く対応できます
   (メニュー「ヘルプ → ログフォルダを開く」から場所を確認できます)

## 音が出ない

- トラックミキサーでミュート(M)がオンになっていないか確認してください
- 全体ミキサーでマスター音量がオフ/0になっていないか確認してください
- Windowsのサウンド設定で、TAB Studioが使用しているオーディオ出力デバイスが
  正しく選択されているか確認してください

## 再生が途中で止まる / カクつく

- 他の重い処理(AI解析など)と同時に再生していないか確認してください
- 曲を読み込み直す(TAB Studioを再起動する)ことで改善する場合があります

## AI採譜が終わらない / 進まない

- 曲の長さやPCの性能によって処理時間が大きく変わります。特に長い曲や、複数の楽器が
  含まれる曲は時間がかかります
- 一定時間待っても進捗が変化しない場合は、アプリを再起動してから再度お試しください
- 改善しない場合は「不具合を報告」からご連絡ください

## プロジェクトファイル(.tabsp)が開けない

- ファイルが破損している、または異なるバージョンのTAB Studioで保存されたファイルの
  可能性があります
- 発生した状況(保存時のバージョン、開こうとした操作)を添えて「不具合を報告」から
  ご連絡ください

## それでも解決しない場合

アプリ内メニュー「ヘルプ → 不具合を報告」から、環境情報を含むテンプレートが
自動生成されます。以下の情報が含まれるので、そのままIssueとして投稿してください:

- TAB Studioのバージョン
- OS情報
- 使用中の音源(VSTi)・エフェクト一覧
- ログファイルの場所

再現手順(どんな操作をしたら発生したか)を可能な限り具体的に記載いただけると、
調査がスムーズに進みます。
