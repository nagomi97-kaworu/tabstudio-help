# VSTプラグイン(音源・エフェクト)

TAB Studio では、TAB譜の各トラックに VST3 形式の音源(VSTi)・エフェクト(FX)
プラグインを割り当てて、その音で再生できます。

## 1. プラグインをスキャンする

メニュー「ファイル → 設定」を開き、「VST3 スキャン」を押すと、お使いのPCの標準的な
VST3インストール先(`C:\Program Files\Common Files\VST3`)がスキャンされ、使用できる
プラグインの一覧が更新されます。

- プラグイン自体は、お使いのDAW等と同様に、あらかじめ各メーカーの手順で個別に
  インストールしておく必要があります(TAB Studioにプラグインは同梱されていません)
- 標準的なインストール先以外にプラグインを入れている場合、現時点ではスキャン対象外です

## 2. トラックに音源(VSTi)を割り当てる

トラックミキサーの「音源(INST)」欄をクリックすると、スキャン済みのVSTiプラグイン
一覧が表示されます。選択するとそのトラックの音がそのプラグインで再生されます。

- 右クリックで「設定を開く」(プラグインのGUI画面を表示)・「アンロード」ができます
- 音源の切り替えは、**完全に停止している状態でのみ**行えます(再生中・一時停止中は
  変更できません)

## 3. トラックにエフェクト(FX)を追加する

トラックミキサーには、1トラックあたり最大3つまでのFXスロットがあります。

- 空きスロットをクリックするとスキャン済みのVSTエフェクト一覧が表示されます
- ロード済みのスロットをクリックするとプラグインの設定画面が開きます
- 右クリックで「設定を開く」「アンロード」ができます

## 4. プロジェクトへの保存

「プロジェクトを保存」すると、各トラックの音源・FXの割り当てとプラグイン内部の
設定(音色やパラメータなど)もあわせて保存されます。次回「プロジェクトを開く」で
同じ状態に復元されます。

> 一部のプラグインでは、内部設定の保存・復元がうまくいかない既知の制約があります。
> 再現する操作手順が分かる場合は「不具合を報告」からお知らせください。

## よくあるトラブル

- **音源が一覧に出てこない**: 「設定 → VST3 スキャン」を実行したか確認してください。
  標準的なインストール先以外にある場合は現時点では検出できません
- **音源を切り替えられない**: 再生中・一時停止中は切り替えできません。一度停止してから
  お試しください
- **保存した音源設定が復元されない**: [トラブルシューティング](troubleshooting.md)も
  あわせてご確認ください
