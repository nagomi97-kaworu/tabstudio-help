# Troubleshooting

## The app won't start / closes immediately

1. Confirm you're running Windows 10/11 (64-bit)
2. Restart your PC and try launching again
3. If the problem persists, use "Report a Bug" and describe when it happens (right at
   startup, or after some action). Attaching the log folder contents helps us respond
   faster (find its location via **Help → Open Log Folder**)

## No sound

- Check that Mute (M) isn't enabled in the Track Mixer
- Check that the master volume isn't off/zero in the Global Mixer
- Check that TAB Studio's audio output device is correctly selected in Windows sound
  settings

## Playback stutters or stops

- Check whether another heavy process is running at the same time
- Reloading the song (restarting TAB Studio) may help

## The project file (.tabsp) won't open

- The file may be corrupted, or may have been saved by a different version of TAB Studio
- Please describe the situation (the version used to save it, what you did to open it)
  via "Report a Bug"

## Still stuck?

Use **Help → Report a Bug** in the app menu. A template including the following
information is generated automatically — paste it into a new Issue:

- TAB Studio version
- OS info
- Loaded VSTi/effect list
- Log file location

Including detailed reproduction steps (what you did before the issue occurred) helps us
investigate faster.
